Crawler MK I

This is a program written in Java. It is designed to go to a specific website that lists schools applying for government funding. Crawler pulls contact information for each school within the state and school year specified by the user, and exports it to a spreadsheet. MK I is a purely command line displayed program that requires number and letter input. (Click to see Code!)
